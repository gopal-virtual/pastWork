Sympose

A revolutionary community engagement platform.

Type Team Project
Role Product & Engineering
Focus Community

Sympose is a community engagement platform designed to help businesses build stronger relationships with their customers. It enables brands to create and manage their own online communities, fostering loyalty and engagement.

The platform provides a suite of tools for moderating discussions, managing user profiles, conducting surveys, and analyzing customer feedback. This empowers businesses to understand customer needs and tailor their strategies accordingly.

Technical Implementation

Designed and setup the complete technical architecture, implemented backend using Python Django, and built the frontend application with React JS. Additionally, configured automated CI/CD deployment pipelines and managed server infrastructure using Docker on AWS EC2.
